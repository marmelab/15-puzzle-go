# 15-puzzle-go

A CLI tool to play the 15-puzzle game, in Go.

## Contributing

### Help

Print all commands available

``` bash
make
```

### Install

Install the dependencies, compile the code and build the docker

``` bash
make install
```

### Run the game

Run the 15-puzzle game

``` bash
make run
```

### Run the tests

Run all tests

```bash
make test
```

### Run the linter

Run the gofmt linter

```bash
make lint
```
